Hybernate has been installed in namespace {{ .Release.Namespace }}.

1. Verify the controller is running:

   kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "hybernate.name" . }}

2. Create a ManagedWorkload to start managing a Deployment:

   cat <<EOF | kubectl apply -f -
   apiVersion: hybernate.io/v1alpha1
   kind: ManagedWorkload
   metadata:
     name: my-app
     namespace: default
   spec:
     target:
       kind: Deployment
       name: my-app
     idlePolicy:
       cpuIdleThreshold: 10
       memoryIdleThreshold: 10
       action: Pause
   EOF

3. Or create a WorkloadPolicy to auto-discover idle workloads in a namespace:

   cat <<EOF | kubectl apply -f -
   apiVersion: hybernate.io/v1alpha1
   kind: WorkloadPolicy
   metadata:
     name: default-policy
     namespace: default
   spec:
     mode: suggest
     targetKinds:
       - Deployment
       - StatefulSet
   EOF

4. Check status:

   kubectl get managedworkloads -A
   kubectl get workloadpolicies -A
